## Build
pxWorlds uses Gradle as build system.

### Requirements
* Git
* Java 13

### Checkout sources
Open a shell and type:
```
git clone https://github.com/tomxpcvx/pxWorlds.git
cd pxWorlds/
```
Use this command to run the source code with gradle:
```
./gradlew build
```
